# Rock Paper Scissors

A Rock Paper Scissors game built with HTML and JavaScript as part of [The Odin Project](https://www.theodinproject.com) Foundations curriculum.

## Live Demo
[View Live](https://yugam200630.github.io/Rock-Paper-Scissors/)

## How to Play
- Choose Rock, Paper or Scissors
- Computer randomly picks its choice
- First to 5 wins!

## Built With
- HTML5
- JavaScript

## What I Learned
- DOM manipulation
- Event listeners
- Writing game logic with conditionals
- Linking external JavaScript files
